# Example App

This is an example FastAPI application demonstrating the `fastapi-hypermedia` package.

## Running with Docker Compose

```bash
docker-compose up --build
```

The app will be available at http://localhost:5000

## Configuration

Set configuration via environment variables or modify `.env` file.

## Features

- Collection+JSON API endpoints
- HTML rendering using Jinja2 templates
- Authentication and authorization
- Workflow management examples
